== Description ==
A very lightweight plugin that records and logs all site access for all users.  Great for documentation, and access history logs.  Records username, IP address along with date and time stamp.  Stores all records in a database that is not editable by the plugin.  IP address is linked for quick access to view the geolocation of each logged IP address to your wordpress instance.  By installing this plugin you agree that you as the Word Press site owner or administrator have collected permission from your users to do so.  This plugin also utilizes geoiptool.com in order to view the geolocation of the particular IP address.  There are no warranties or guarantees of accuracy of location or ip information.  

= Features =
* Records logins for all users
* Records date and time stamp for all users
* Records IP address for all users
* Data stored in MySQL Table
* Plugin does not allow editing of the login records
* IP Addresses are linked for easy click to view the geolocation of each logged in event
* Tracker where your users log in from and how often



= Track Your Users Better =
You are able to track all users that log into your Word Press instance.  You can search the login records by username or by IP address.

== Installation ==
1. Upload the plugin files to the '/wp-content/plugins' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
